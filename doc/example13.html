Exporting ontologies into different formats

  1. Export a given ontology to OWL format (oboInOwl mapping).
  2. Export a given ontology to GML format.
  3. Export a given ontology to XML format.
  4. Export a given ontology to DOT format.
  5. Export a given ontology to OBO format (spec 1.2).
  6. Export a given ontology to RDF format.
  1. Export a given ontology to OWL format (oboInOwl mapping): 
    					
			use OBO::Parser::OBOParser;
			use strict;
						
			my $my_parser = OBO::Parser::OBOParser->new();
			
			my $o = $my_parser->work("gene_ontology.obo");
			
			# export to OWL: gene_ontology.owl
			open (FH, ">gene_ontology_by_apo.owl");
			$o->export('owl', \*FH, \*STDERR, 'http://www.cellcycleontology.org/ontology/owl/', 'http://www.cellcycleontology.org/formats/oboInOwl#');
			close FH;
